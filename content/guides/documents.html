Doris 文档

1. Install 编译与安装

该文档介绍了如何通过源码编译和安装 Doris，并介绍了最基本的部署方式，以及编译和部署过程中可能遇到的问题。同时提供了预编译版本的下载连接。这是您使用 Doris 的第一步。

2. Deploy & Upgrade 部署与升级

该文档详细介绍了 Doris 部署与升级的步骤以及可能遇到的问题。如果您希望将 Doris 部署于生产环境。强烈建议仔细阅读这个文档。

3. Getting start 基础使用指南

该文档可能帮助您快速体验整个 Doris 使用流程。非常适合第一次使用 Doris 的用户进行上手体验。

4. Best Practices 最佳实践

该文档简要介绍了 Doris 的一些独有特性和功能，可能帮助您初步了解 Doris 的特性。

5. Doris Create, Load & Delete 建表、导入和删除

该文档详细介绍了 Doris 在建表、数据导入和删除数据过程中的使用注意事项。可以帮助您详细了解 Doris 的数据流。

6. Data Model, Rollup & Prefix Index 数据模型、ROLLUP和前缀索引

该文档介绍了 Doris 最核心的部分：数据模型。建立一个合适的数据模型将最大程度发挥 Doris 的优势。如果您希望进一步使用 Doris，强烈建议仔细阅读这个文档。

7. Metadata 元数据

该文档介绍了 Doris 元数据的一些实现细节。您在阅读完该文档后，再配合 Metadata Operations 元数据运维 文档，可以更好的对 Doris 集群进行维护和管理。
